# 📏 Screen Ruler - Расширение для Chrome

**Профессиональный инструмент для измерения элементов и расстояний на веб-страницах**

Screen Ruler - это мощное браузерное расширение, специально разработанное для веб-разработчиков, веб-дизайнеров и всех, кто работает с веб-интерфейсами. Позволяет точно измерять размеры элементов страницы и расстояния между ними.

## 🎯 Основные возможности

### ✅ **Измерение элементов**
- Подсвечивание элементов при наведении курсора
- Отображение точных размеров (ширина × высота)
- Показ box-model (content, padding, margin)
- Информация о селекторе элемента

### ✅ **Измерение расстояний**
- Фиксация элементов для измерения расстояний
- Визуальные линии между элементами
- Точные значения в пикселях
- Поддержка горизонтальных и вертикальных расстояний

### ✅ **Удобный интерфейс**
- Гибридный подход: hover-окно + информация на обрамлении
- Умное позиционирование (не выходит за границы экрана)
- Перетаскиваемая панель управления
- Адаптивная цветовая схема

### ✅ **Настройки**
- Интерактивный и неинтерактивный режимы
- Показ позиции курсора
- Горячие клавиши
- Сохранение настроек между сайтами

## 🚀 Установка

### Установка в режиме разработчика

1. **Скачайте исходный код**
   ```bash
   git clone https://github.com/yourusername/screen-ruler.git
   cd screen-ruler
   ```

2. **Откройте Chrome и перейдите в управление расширениями**
   - Введите в адресной строке: `chrome://extensions/`
   - Или через меню: ⋮ → Дополнительные инструменты → Расширения

3. **Включите режим разработчика**
   - Активируйте переключатель "Режим разработчика" в правом верхнем углу

4. **Загрузите расширение**
   - Нажмите кнопку "Загрузить распакованное расширение"
   - Выберите папку с расширением Screen Ruler
   - Расширение появится в списке

5. **Закрепите расширение**
   - Нажмите на иконку расширений в панели инструментов
   - Найдите Screen Ruler и нажмите на значок 📌 для закрепления

### Установка из Chrome Web Store

## 📖 Инструкция по использованию

### 🎮 **Базовое использование**

1. **Активация расширения**
   - Кликните на иконку Screen Ruler в панели инструментов
   - Нажмите переключатель "Расширение" в popup
   - Или используйте кнопку "Включить" в контрольной панели

2. **Измерение элементов**
   - Наведите курсор на любой элемент страницы
   - Элемент подсветится красной рамкой
   - Появится hover-окно с информацией:
     - Селектор элемента (div#header.container)
     - Размеры (320 × 240 px)
     - Box-model (content, padding, margin)

3. **Измерение расстояний**
   - Наведите курсор на первый элемент
   - Нажмите `Alt` или кликните левой кнопкой мыши
   - Элемент зафиксируется (изменит цвет на бирюзовый)
   - Наведите курсор на второй элемент
   - Между элементами появятся линии с расстояниями

### ⌨️ **Горячие клавиши**

| Клавиша | Действие |
|---------|----------|
| `Alt` | Зафиксировать элемент под курсором |
| `Escape` | Снять фиксацию элемента |
| `F12` | Открыть DevTools (стандартная функция) |

### 🎛️ **Панель управления**

Панель управления появляется в правом верхнем углу страницы:

- **Индикатор статуса**: Показывает состояние расширения
  - 🟢 Зеленый - активно
  - 🟡 Желтый - на паузе
  - ⚪ Серый - отключено

- **Кнопка "Пауза"**: Временно отключает функциональность
- **Кнопка "⚙️"**: Открывает popup с настройками

### 🔧 **Настройки**

Доступны в popup расширения:

#### **Основные настройки**
- **Показывать позицию курсора**: Отображает координаты X,Y в левом нижнем углу
- **Интерактивный режим**: Разрешает обычные клики по элементам сайта

#### **Режимы работы**
- **Интерактивный**: События сайта работают как обычно
- **Неинтерактивный**: Клики перехватываются расширением

## 🎨 **Визуальная схема работы**

### Подсвечивание элементов
```
┌─ div#header ─ 320×240 ──┐ ← Информация на обрамлении (фиксированный)
│                        │
│     Element Content    │
│                        │
└────────────────────────┘

┌─────────────────────────┐
│ div#header.container    │ ← Hover-окно (наведение)
├─────────────────────────┤
│ 320 × 240 px           │
├─────────────────────────┤
│ Content: 280×200       │
│ Padding: 20px          │
│ Margin: 15px           │
└─────────────────────────┘
```

### Измерение расстояний
```
┌─────────────┐     50px     ┌─────────────┐
│  Element A  │ ◄─────────► │  Element B  │
│ (фиксирован)│             │ (наведение) │
└─────────────┘             └─────────────┘
```

## 🛠️ **Техническая информация**

### Архитектура
- **Manifest V3**: Современная архитектура расширений Chrome
- **Content Script**: Основная логика работы с DOM
- **Background Script**: Управление состоянием и настройками
- **Popup Interface**: Интерфейс управления

### Модульная структура
```
Screen Ruler/
├── Core Engine (измерения, вычисления)
├── Element Selector (выбор и подсвечивание)
├── UI Manager (hover-окно, элементы интерфейса)
├── Event Manager (обработка событий)
├── Settings Manager (настройки)
└── Distance Calculator (расстояния)
```

### Безопасность
- Минимальные permissions (activeTab, storage)
- Изоляция стилей через CSS
- Безопасная обработка DOM элементов
- Защита от XSS и injection атак

## 🐛 **Устранение проблем**

### Расширение не работает
1. Убедитесь, что расширение включено в `chrome://extensions/`
2. Проверьте, что кнопка включения активна в popup
3. Обновите страницу после включения расширения

### Элементы не подсвечиваются
1. Снимите расширение с паузы
2. Проверьте, что страница полностью загружена
3. Попробуйте обновить страницу

### Hover-окно не отображается
1. Убедитесь, что элемент не является частью расширения
2. Проверьте, что курсор действительно находится на элементе
3. Попробуйте навести на другой элемент

### Горячие клавиши не работают
1. Убедитесь, что фокус находится на странице (не в DevTools)
2. Проверьте, что расширение не на паузе
3. Попробуйте нажать клавишу еще раз
